# אפיון תפקודי - ReagentItem

## מטרה
רכיב תצוגה של ריאגנט בודד - מציג את כל המידע החשוב בצורה ויזואלית וברורה.

## תיאור למשתמש
כל ריאגנט מוצג בכרטיס שמכיל:
- 🧪 שם הריאגנט
- 📊 כמות כוללת ממוצע (total_quantity_all_batches)
- 📅 תאריך תפוגה קרוב ביותר
- 🏢 ספק
- 🎯 קטגוריה (reagents/cells/controls)
- ⚠️ סטטוס מלאי (in_stock/low_stock/out_of_stock)
- 📦 מספר אצוות פעילות
- 📈 חודשי מלאי נותרים

## UI Elements

### Visual Indicators:
- 🟢 ירוק: מלאי תקין (> 2 חודשים)
- 🟡 צהוב: מלאי נמוך (< 2 חודשים)
- 🔴 אדום: אזל מלאי או פג תוקף קרוב
- 📛 Badge: "פג תוקף בעוד X ימים"

### Interactions:
- לחיצה על כרטיס → פרטים מלאים
- Hover → tooltip עם מידע נוסף
- Quick Actions: ערוך, הזמן, צפה באצוות

## מקרי שימוש

### UC1: סריקה מהירה של מצב מלאי
1. נכנס לעמוד ריאגנטים
2. רואה רשימת כרטיסים
3. מזהה מיד בצבעים מה דורש תשומת לב
4. 5 כרטיסים אדומים = בעיות

### UC2: לחיצה על כרטיס לפרטים
1. לוחץ על Anti-A
2. רואה את כל האצוות
3. רואה היסטוריית הזמנות
4. רואה צריכה חודשית

## כללי עסקיים

1. **Color Coding**:
   - out_of_stock → אדום
   - low_stock → כתום
   - in_stock → ירוק

2. **Expiry Warning**:
   - < 14 days → badge אדום
   - < 30 days → badge כתום

3. **Action Buttons**:
   - אם low_stock → "הזמן כעת"
   - אם missing COA → "העלה COA"
   - תמיד: "ערוך", "צפה באצוות"
