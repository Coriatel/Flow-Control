# אפיון תפקודי - alertsEngine

## מטרה
מנוע ההתראות - בודק את כל המערכת לפי כללים ויוצר התראות.

## סוגי התראות
1. 🚨 פג תוקף (< 14 ימים)
2. 📉 מלאי נמוך (< 1 חודש)
3. ⚫ אזל מלאי (0)
4. ⏰ הזמנה מאוחרת
5. 📄 אצווה חסרת COA
6. 📅 ספירה מאוחרת (> 45 ימים)

## תהליך
1. רץ כל שעה/יום (scheduled)
2. בודק כל AlertRule פעילה
3. אם התנאי מתמלא → יוצר ActiveAlert
4. אם התנאי לא מתמלא → auto-resolve
5. שולח notifications (email/dashboard)
