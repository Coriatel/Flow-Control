# אפיון תפקודי - getEditOrderData

## מטרה
טעינת כל נתוני ההזמנה לעריכה - ההזמנה, הפריטים, המשלוחים, והמשיכות.

## תיאור
- 📋 ההזמנה המלאה
- 📦 כל הפריטים עם סטטוס
- 🚚 משלוחים שהתקבלו
- 📤 משיכות שהוגשו (מסגרת)
- ✏️ עריכת פרטים/כמויות/SAP

## מקרי שימוש

### UC1: השלמת פרטי SAP
1. הזמנה ב-status: pending_sap_details
2. עורך
3. ממלא order_number_permanent + purchase_order_number_sap
4. שומר
5. סטטוס → approved

### UC2: עריכת כמויות
1. הזמנתי 10 יח'
2. צריך 15
3. עורך
4. משנה 10→15
5. quantity_remaining מתעדכן

### UC3: בדיקת משלוחים שהתקבלו
1. עורך הזמנה
2. רואה ש-3 מתוך 5 פריטים התקבלו
3. לוחץ על משלוח → פירוט
