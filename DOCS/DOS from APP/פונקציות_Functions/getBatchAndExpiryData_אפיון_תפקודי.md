# אפיון תפקודי - getBatchAndExpiryData

## מטרה
טעינת כל האצוות עם סטטוס תפוגה, FIFO tracking, והמלצות טיפול.

## תיאור
- 🚨 אצוות פגות/פגות תוקף
- 📅 אצוות ישנות (FIFO)
- ✅ מה כבר טופל (ExpiredProductLog)
- 📊 סטטיסטיקות: כמה פג היום, השבוע, החודש
- 💡 המלצות פעולה (השמדה, שימוש אחר)

## מקרי שימוש

### UC1: בדיקת פגי תוקף
1. נכנס לניהול פגי תוקף
2. רואה 5 אצוות פגות היום
3. רואה 12 פגות השבוע
4. לכל אחת - המלצת פעולה

### UC2: FIFO - אצוות ישנות
1. רואה אצוות בנות 6+ חודשים
2. ממליץ להשתמש ראשון
3. סימון בצבע
