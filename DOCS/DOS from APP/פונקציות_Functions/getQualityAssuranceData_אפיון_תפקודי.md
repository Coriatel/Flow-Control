# אפיון תפקודי - getQualityAssuranceData

## מטרה
טעינת כל נתוני בקרת האיכות - COAs, QC status, אצוות בהסגר, action items.

## תיאור
- 🔬 כל האצוות עם QC status
- 📄 מעקב COAs (יש/חסר)
- 🚧 אצוות בהסגר
- ✅ QC passed/failed
- ⚠️ Action required

## מקרי שימוש

### UC1: בדיקת אצוות בהסגר
1. נכנס לבקרת איכות
2. רואה 5 אצוות בהסגר
3. לכל אחת - כמה ימים בהסגר
4. מה חסר (COA? QC?)

### UC2: העלאת COA
1. רואה אצווה חסרת COA
2. לוחץ "העלה COA"
3. מעלה קובץ
4. הסטטוס מתעדכן
