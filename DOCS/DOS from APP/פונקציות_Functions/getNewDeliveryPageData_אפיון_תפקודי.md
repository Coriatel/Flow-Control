# אפיון תפקודי - getNewDeliveryPageData

## מטרה
טעינת כל האופציות הזמינות לקליטת משלוח חדש - הזמנות פתוחות, ריאגנטים לפי ספק, אצוות קיימות.

## תיאור למשתמש
כשנכנס לקליטת משלוח:
- 📋 בוחר ספק → רואה את כל ההזמנות הפתוחות שלו
- 🎯 בוחר הזמנה → רואה את הפריטים שהוזמנו
- ✅ מסמן פריטים שהגיעו
- 📝 ממלא batch number, תפוגה, כמות
- 💾 שומר

## מקרי שימוש

### UC1: קליטת משלוח עם הזמנה
1. בוחר ספק BIORAD
2. רואה 3 הזמנות פתוחות
3. בוחר REQ-123
4. רואה 5 פריטים בהזמנה
5. מקליט 3 מתוכם

### UC2: קליטת משלוח ללא הזמנה
1. בוחר "ללא הזמנה"
2. בוחר ריאגנטים מהרשימה
3. ממלא פרטים
4. שומר
