# אפיון תפקודי - exportAllCoas

## מטרה
ייצוא כל תעודות האנליזה (COAs) ל-ZIP מאורגן - לביקורת או גיבוי.

## תהליך
1. אוסף את כל ה-COAs
2. מארגן לפי ספק/ריאגנט
3. יוצר מבנה תיקיות
4. מוריד את כל הקבצים
5. יוצר ZIP
6. מחזיר להורדה

│       └── GHI789_2025-09-20.pdf
