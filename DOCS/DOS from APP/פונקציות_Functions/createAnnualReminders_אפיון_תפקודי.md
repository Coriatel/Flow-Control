## מטרת הפונקציה

יצירת תזכורות אוטומטיות לפעולות שנתיות: ביקורת, גיבוי, ארכוב, חידוש רישיונות.

## תזכורות

1. **Annual Audit**: ביקורת שנתית (ינואר)
2. **COA Archive**: ארכוב COAs (דצמבר)
3. **Data Cleanup**: ניקוי שנתי (דצמבר)
4. **License Renewal**: חידוש רישיונות (לפי תאריך)
