# אפיון תפקודי - getManageSuppliersData

## מטרה
טעינת ספקים עם סטטיסטיקות - כמה ריאגנטים, הזמנות, וביצועים.

## תיאור
- 🏢 כל הספקים
- 📊 כמה ריאגנטים לכל ספק
- 📈 כמה הזמנות פעילות
- ⭐ ביצועי ספק

## סטטיסטיקות לספק

1. **Reagents Count**: כמה ריאגנטים מהספק
2. **Active Orders**: הזמנות פעילות
3. **Avg Delivery Time**: זמן אספקה ממוצע
4. **Total Value YTD**: סכום השנה
