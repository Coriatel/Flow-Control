# אפיון תפקודי - calculateAverageUsage

## מטרה
חישוב צריכה חודשית ממוצעת מ-InventoryTransactions - הבסיס לכל החישובים במערכת.

## הלוגיקה
```
6 חודשים אחרונים:
- ינואר: 20 יח' נמשכו
- פברואר: 25 יח'
- מרץ: 18 יח'
- אפריל: 22 יח'
- מאי: 24 יח'
- יוני: 21 יח'

סה"כ: 130 יח' ב-6 חודשים
ממוצע: 130/6 = 21.67 יח'/חודש
```

## שימוש
- רץ אוטומטית כל לילה
- או ידנית מדף ניהול נתוני צריכה
- מעדכן Reagent.average_monthly_usage
